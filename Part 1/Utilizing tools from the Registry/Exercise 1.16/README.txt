Deploy Sample App using Python and streamlit. This is growth calculator that determines the worth of an investment given the growth rate, initial investment, and time in years.

1. Need to use python:3.8
2. Create a working directory /home/app/webapp and set in ENV variable MICRO_SERVICE
3. Set an environment variables (PYTHONDONTWRITEBYTECODE , PYTHONUNBUFFERED) both is equal to 1.
4. This project requires Python libraries to be installed for it to run. The libraries will be recorded in a requirements.txt file.
5.Copy project.For ease of handling during the Docker building, place both the Python and requirements files in a folder and name it src/. 
6. This app is use port 8501
6. Run the Streamlit app ( streamlit run app.py )
